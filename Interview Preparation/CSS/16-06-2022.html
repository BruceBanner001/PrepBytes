Q1.Difference between Flex Row and Column ?

Default Flex Direction is row

This is a CSS Property on the concept of Flex Module. The flex-direction property specifies the direction of the flexible items.
Flex Row:
The Child Containers will be placed Horizontally accroding to the Parent Container.
Flex Column:
The Child Containers will be placed Vertically accroding to the Parent Container.
Note: First we need to define display as Flex in Parent Container. Otherwise, it won't work. 

                <div class="container">
                    <div class="child">One</div>
                    <div class="child">One</div>
                    <div class="child">One</div>
                </div>

Q2. Explain Inline, Internal and External Style Sheets ?

Inline:
An inline CSS is used to apply an Unique Style to a Single HTML Element.

Internal: It also add Unique Style to same type of HTML Elements or Class or ID in Single HTML File. We define Internal CSS inside an Tag.

External: It is also same as Internal CSS but with one difference. External means we save styles in different file end with .css. We can use this External Styles as much as want in any number of HTML files. In order to use that we have to link the CSS File using link tag.

Q3.Justify-content allows you to do what?

 This is a CSS Property on the concept of Flex Module. It defines the browser how to distribute the space between or around contents accroding to the Parent Container.

Q4.Difference between absolute and relative positioning ?

Position: relative:
Places an element relative to its current position without changing the layout around it.
Position: absolute:
places an element relative to its parent's position and changing the layout around it.

Q5.What is grid-template columns used for ?

 The grid-template-columns property specifies the number (and the widths) of columns in a grid layout. The values are a space separated list, where each value specifies the size of the respective column.

Q6.What is the z-index in CSS ?

 The z-index property specifies the stack order of an element. An element with greater stack order is always in front of an element with a lower stack order.
 Only works with Positions Only. We can also use Positive and Negative Values.

Q7. Difference between Margin and Padding ?

These are CSS Elements.We can provide space between elements of the four sides top, bottom, right and left. Margin will applied outside the border. Padding will applied inside the border. We can use values in like %, px, vh, vw, etc.
